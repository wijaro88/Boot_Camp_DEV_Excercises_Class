Condicionales.

Introducción

Las condicionales van a ser estructuras de control que nos van a permitir con base en una condición o más decidir qué lineas de código se ejecutan.

Operadores relacionales

Un operador relacional, nos va a permitir evaluar dos elementos, para entender la relación entre ellos, si el resultado de la comparación es correcto la expresión considerada es verdadera, en caso contrario es falsa. “Verde” !== “Verde” => false

  • ==: Comparación de dos elementos para saber si son iguales sin importar el tipo de dato.
  • ===: Comparación de dos elementos para saber si son iguales hasta en el tipo de dato.
  • !=: Comparación de dos elementos para saber si son diferentes y no en el tipo de dato.
  • !==: Comparación de dos elementos para saber si son diferentes hasta en el tipo de dato.
  • >: Comparación de dos elementos para saber si el primero es más grande que el segundo.
  • >=: Comparación de dos elementos para saber si el primero es más grande o igual que el segundo.
  • <: Comparación de dos elementos para saber si el primero es más pequeño que el segundo.
  • <=: Comparación de dos elementos para saber si el primero es más pequeño o igual que el segundo.
